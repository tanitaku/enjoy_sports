# 主な機能
<ul>
 <li>ユーザー(user), ポスト(post), のCRUD</li>
 <li>ログイン、ログアウト、新規登録</li>
 <li>ユーザーフォロー機能</li>
 <li>コメント機能</li>
 <li>都道府県名検索機能</li>
 <li>管理者機能（ユーザーの追加・編集・削除）</li>
</ul>
